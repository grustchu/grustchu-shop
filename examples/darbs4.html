Bootstrap

Bootstrap ir milzīgs parocīgu, atkārtoti lietojamu koda bitu kolekcija, kas rakstīta HTML, CSS un JavaScript. Tas ir arī priekšgala izstrādes ietvars, kas ļauj izstrādātājiem un dizaineriem ātri izveidot pilnībā atsaucīgas vietnes. Būtībā Bootstrap ietaupa jūs no daudz CSS koda rakstīšanas, dodot jums vairāk laika, ko veltīt tīmekļa lapu izstrādei.
